# Project Vision — FyT Lab Connect

## Visión
Ser la plataforma de referencia para la difusión y colaboración académica del Grupo FyT, sirviendo igualmente como base técnica sólida para extensión hacia servicios institucionales y comerciales, manteniendo principios éticos y científicos.

## Problema que resuelve
La dispersión de información académica y la falta de un punto centralizado para publicar y coordinar actividades de investigación limitan visibilidad y colaboración.

## Propuesta de valor
- Un portal público que organiza la producción académica del grupo.
- Interfaces intuitivas para navegar publicaciones, proyectos y eventos.
- Base técnica preparada para escalar (SSG/SSR, CI/CD, accesibilidad).

## Proyección futura
- Mantener la plataforma como recurso académico primario.  
- Evolucionar hacia servicios opcionales para instituciones y terceros (previa validación legal/comercial).
